# Что такое Hysteria-Stealth?

**Hysteria-Stealth — это форк оригинального Hysteria 2, который добавляет многослойную обфускацию трафика для противодействия современным системам Deep Packet Inspection (DPI). В то время как стандартный Hysteria 2 уже маскируется под HTTP/3, Hysteria-Stealth идёт дальше, внедряя поведенческие, временные и контентные паттерны, которые делают трафик неотличимым от легитимного пользовательского трафика.**
